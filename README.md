## Описание
Проектная работа курса «Веб-разработчик» от «Яндекс Практикум».
Одностраничный сайт, реализованный с помощью HTML и CSS. Построен по технологии БЭМ Nested и Flexbox.

### Технологии
* HTML 5
* CSS 3
* БЭМ Nested
* Flexbox
* Keyframes
* Iframe
* Позиционирование

#### Планы по доработке проекта
* Адаптивная верстка
